**Technologies:**

*   Yarn
*   Lerna
*   Babel
*   ESLint
*   Prettier

## Development

Clone and install dependencies:

```sh
git clone https://github.com/mrpatiwi/monorepo-starter.git
cd monorepo-starter
yarn
```

> Change `private` attribute to `false` on the packages.json to be able to push to npm.
